# AGENTS.md

This project uses the **Stateful Spec** methodology for AI-assisted development.
See `.stateful-spec/methodology/overview.md` for the full methodology.

## Project Context

The Project Definition is at `.stateful-spec/project-definition.md`. Load it at the start of every session.

## OpenCode Commands

Available slash commands in `.opencode/commands/`:

| Command | Description |
|---------|-------------|
| `/resume-session` | Load project context and pick up where you left off |
| `/save-session` | Save session progress to memory.md and iteration files |
| `/create-technical-spec` | Create a technical specification for new work |
| `/write-tests` | Write tests for existing or new code |
| `/debug-issue` | Diagnose and fix a bug with root cause analysis |
| `/refactor-code` | Restructure code without changing external behavior |
| `/review-changes` | Self-review code changes before committing |
| `/write-commit-message` | Generate a structured commit message |
| `/update-documentation` | Update project docs after implementing changes |
