# Changelog Updates

When making changes to the Twenty CRM LinkedIn Extension, remember to update the CHANGELOG.md file.

## When to Update

Update `CHANGELOG.md` when:

- Adding new features
- Fixing bugs
- Changing existing behavior
- Updating dependencies significantly
- Modifying the UI/UX

## Changelog Format

Follow this format in CHANGELOG.md:

```markdown
## [VERSION] - YYYY-MM-DD

### ✨ Features

- **Feature Name** - Description

### 🐛 Bug Fixes

- Fixed issue with X

### 🔧 Technical

- Internal change description

### 📋 Data Captured

- New data fields if applicable
```

## Example Entry

```markdown
## [1.1.0] - 2024-12-18

### ✨ Features

- **New Feature** - Added ability to do X

### 🐛 Bug Fixes

- Fixed LinkedIn profile image not uploading correctly
```

## Reminder

After completing feature work, ask: "Should I update the CHANGELOG.md with these changes?"
