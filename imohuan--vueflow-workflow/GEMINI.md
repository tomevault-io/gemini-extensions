# Git 提交和推送工作流

## 目的

规范化项目的 Git 提交和推送流程。必须使用 PowerShell 命令

## 工作流程

### 1. 提交更改

执行以下命令提交所有更改到本地仓库：

#### PowerShell 命令

```powershell
git add .; git commit -m "描述性的中文提交信息"; git push
```
