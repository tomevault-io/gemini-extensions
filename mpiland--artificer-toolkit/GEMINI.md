## artificer-toolkit

> **"FocusForge "** - An advanced Pomodoro timer application with AI-powered productivity insights

## Project Context
### Project Description
**"FocusForge "** - An advanced Pomodoro timer application with AI-powered productivity insights

### Tech Stack
- Primary framework: React/Tanstack Start
- State Management: Zustan
- Database: Supabase
- DO NOT USE: jQuery, Angular, Emotion CSS
- CSS: Vanilla Extract CSS
- Schema: Zodv4
