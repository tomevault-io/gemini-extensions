# social-media-kit

> Tome by [terrytangyuan](https://github.com/terrytangyuan/social-media-kit) — distributed by [TomeVault](https://tomevault.io)

A comprehensive social media management tool designed to help you create, format, and post content across multiple platforms including LinkedIn, Twitter/X, Bluesky, and Mastodon. Features advanced styling, tagging, reminder, OAuth authentication, and cross-platform posting capabilities. Free and open source.
