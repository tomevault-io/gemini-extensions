# mozaiks

> Source: [BlocUnited-LLC/mozaiks](https://github.com/BlocUnited-LLC/mozaiks) — distributed by [TomeVault](https://tomevault.io)

The Mozaiks core is a self-hostable runtime for deploying AI-driven, multi-tenant web apps with real-time UI, auth, and billing built in. Apps are defined via YAML and lightweight Python/JS stubs for agent logic, workflows, and integrations — so developers focus on behavior, not plumbing. :link: https://docs.mozaiks.ai/
