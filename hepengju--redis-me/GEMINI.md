> Git 提交信息仅标题、保持简洁


# Git 提交约定

- **只写一行标题**（subject），**不要**多段正文（body）。
- 标题保持简短，沿用常见前缀即可，例如：`feat(scope): 简述`。
- 为用户执行 `git commit` 时：使用 `git commit -m "标题"`，不要加第二个 `-m` 写正文。
