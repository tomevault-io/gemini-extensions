# xiaozhi-client-flutter

> Tome by [eiriksgata](https://github.com/eiriksgata/xiaozhi-client-flutter) — distributed by [TomeVault](https://tomevault.io)

小智AI 一款基于 Flutter 开发的跨平台 AI 聊天助手应用，支持多智能体管理、文字/语音多模态交互，支持按键语音说话，支持实时打断语音对话（前端使用AEC技术，后台需要支持打断），采用 WebSocket 实时通信。只需要登录小智官网 添加即可使用。提供动态配置，方便你本地调试不同的小智AI后台。
