# RallyClip

> Source: [iroblesrazzaq/RallyClip](https://github.com/iroblesrazzaq/RallyClip) — distributed by [TomeVault](https://tomevault.io)

RallyClip is a free and open source tennis match segmentation tool that takes a raw tennis match video and removes all dead time between points, producing a segmented video containing only the points.
