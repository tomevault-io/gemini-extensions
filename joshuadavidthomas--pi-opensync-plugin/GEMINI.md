# Agent Guidelines

Pi coding agent extension that syncs sessions to OpenSync dashboards.

## Commands

```bash
bun install        # Install dependencies
bun run test       # Run tests
bun run typecheck  # Type check
```

## Development

Hot reload: edit `src/*.ts`, then `/reload` in pi.
