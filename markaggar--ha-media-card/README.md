# ha-media-card

> Tome by [markaggar](https://github.com/markaggar/ha-media-card) — distributed by [TomeVault](https://tomevault.io)

A Home Assistant front end card that can show a specific photo or video, or a sequential or randomized slide show from an entire folder structure. Full UI config with media browsing built in, plus a ton of other smart features. Also integrate with ha-media-index for metadata support and smart actions.

## Gemini CLI Config

The `GEMINI.md` file in this directory is the project config converted for Gemini CLI.
Original source: `copilot-instructions.md` in [markaggar/ha-media-card](https://github.com/markaggar/ha-media-card).
