# AGENTS.md

## What This Is

11 go-to-market skills for B2B SaaS companies — built for founders of AI and technical products, GTM leaders, and operators from Seed through Series D. Each skill is a self-contained `SKILL.md` in the `skills/` directory.

## How to Use These Skills

Skills are loaded based on what the user is working on. Here's when to load each one:

**Positioning & pricing questions** — user is struggling with messaging, competitive differentiation, or pricing architecture:
- `positioning-strategy` → messaging sounds like competitors, category creation, repositioning
- `technical-product-pricing` → usage-based vs seat-based debates, freemium gates, price increases
- `ai-gtm` → selling AI products, "will this replace my team?" objections, AI pricing models

**Building GTM motions** — user is setting up or scaling go-to-market channels:
- `partnership-architecture` → partner programs, channel strategy, certification wedges, build-vs-partner
- `developer-ecosystem` → developer adoption, open vs curated marketplaces, student programs
- `product-led-growth` → self-serve funnels, PLG vs sales-led decisions, growth experimentation
- `0-to-1-launch` → new market entry, first 50 customers, partner-led launches

**Enterprise execution** — user is in active deals or post-sale:
- `enterprise-account-planning` → stuck deals, MEDDICC qualification, procurement navigation, mutual action plans
- `enterprise-onboarding` → champion went quiet, adoption stalled, implementation planning

**Running the company** — user needs operating rhythm or board prep:
- `operating-cadence` → meeting structures, metric dashboards, quarterly planning, decision rights
- `board-and-investor-communication` → board decks, investor updates, missed quarter narratives

## What Makes These Skills Different

Each skill contains specific decision thresholds (not principles), real stories where things went wrong before they went right, and the counter-intuitive patterns most operators learn too late. When applying a framework, use the specific numbers and criteria in the skill — they're earned, not theoretical.
