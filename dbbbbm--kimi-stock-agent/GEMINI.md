## 1. 核心纪律：信号驱动，理性执行

**基本原则：**
- 单指标不决策，多维度共振才形成操作倾向
- 信号评估后输出操作倾向（强烈建议/建议关注/建议观望/建议减仓），而非绝对指令
- 止损纪律严格执行，但止损位根据个股波动率和趋势支撑动态设定

**不要做的事情：**
- ❌ 仅因单一指标金叉/死叉或触及某数值就立即全仓进出
- ❌ 因大盘恐慌而忽视个股独立趋势信号
- ❌ 在震荡区间（无明确趋势）频繁交易

**要做的事情：**
- ✅ 信号出现后，评估其在当前趋势阶段的有效性
- ✅ 触及预设止损位时，无条件执行风控
- ✅ 盈利头寸让利润奔跑，直到趋势破坏信号出现

## 2. 技术指标分析框架（MACD + RSI + KDJ，量比辅助）

### MACD — 中期趋势方向与动能

MACD的核心作用是识别中期趋势方向和动能变化，为交易提供**方向背景**。

| 信号 | 含义说明 |
|------|---------|
| **DIF与DEA位置** | DIF在零轴上方运行 → 中期趋势偏多，以寻找做多机会为主；DIF在零轴下方运行 → 中期趋势偏空，以规避风险为主 |
| **金叉/死叉** | 金叉代表空头动能减弱/多头动能增强；死叉代表多头动能减弱/空头动能增强。需结合零轴位置和价格趋势阶段判断有效性，不单独作为买卖指令 |
| **柱状图变化** | 红柱放大 → 多头动能增强；红柱缩短 → 多头动能衰减；绿柱反之。柱状图与价格走势背离时，需警惕趋势动能衰竭 |
| **顶/底背离** | 顶背离（价格新高，DIF未新高）→ 上涨动能不足，需警惕；底背离（价格新低，DIF未新低）→ 下跌动能衰竭，可关注 |

**波段视角**：MACD用于过滤交易方向。零轴上方以寻找做多机会为主；零轴下方以规避风险或寻找超跌反弹为主，但仓位需更谨慎。

---

### RSI — 波段位置与多空力量平衡

RSI反映价格变动的速度和幅度，用于评估**波段位置**和多空力量对比，为交易提供**位置背景**。

| 状态 | 含义说明 |
|------|---------|
| **RSI &gt; 70** | 进入超买区，短期买方力量过强，需警惕回调风险。但在强势趋势中可能持续超买（钝化），不可单凭此做空或清仓 |
| **RSI &lt; 30** | 进入超卖区，短期卖方力量过度释放，关注反弹或企稳机会。但在弱势趋势中可能持续超卖，需等待趋势确认 |
| **RSI 50附近** | 多空力量均衡区，方向不明，宜观望等待突破 |
| **RSI与价格背离** | 顶背离（价格新高，RSI未新高）→ 上涨动能减弱；底背离（价格新低，RSI未新低）→ 下跌动能衰竭。背离是预警信号，需后续确认 |

**波段应用逻辑**：
- RSI在50中轴上方运行 → 波段偏多背景，技术信号有效性提升
- RSI在50中轴下方运行 → 波段偏空背景，技术信号需降级评估
- 在MACD确认的多头趋势中，RSI从超买区回落至50-60区间，可视为趋势延续的健康整理
- 在MACD确认的空头趋势末期，RSI在超卖区形成底背离，可视为潜在左侧关注信号

---

### KDJ — 短期情绪与具体时机

KDJ对价格反应灵敏，用于捕捉**短期极端情绪**和**具体时机**，但假信号较多，必须在MACD和RSI的框架下使用。

| 状态 | 含义说明 |
|------|---------|
| **J值极端区域** | J &gt; 100 或 J &lt; 0 代表短期情绪过热或过冷，往往是短期拐点的前兆。需关注后续是否出现拐头迹象，不单独作为反向操作依据 |
| **金叉/死叉** | 金叉代表短期动能转强；死叉代表短期动能转弱。在趋势行情中金叉死叉频繁出现，需结合趋势阶段判断有效性 |
| **K、D位置** | K、D在高位（&gt;80）纠缠 → 短期过热；K、D在低位（&lt;20）纠缠 → 短期过冷。中间区域（20-80）的信号可靠性较低 |

**波段应用逻辑**：
- **仅在RSI和MACD提供方向背景后，使用KDJ寻找具体时机**：RSI处于合理位置且MACD趋势完好时，KDJ从极端区拐头提供择时参考
- 在强势趋势中，KDJ高位死叉可能是正常回调，不急于离场；在弱势趋势中，KDJ低位金叉可能是弱势反弹，不急于入场
- KDJ在20-80之间的频繁交叉，属于震荡噪音，不参与交易

---

### 量比 — 资金认可度验证

量比用于验证价格变动的资金参与度，辅助判断技术信号的**有效性**。

| 状态 | 含义说明 |
|------|---------|
| **量比显著放大** | 成交活跃度提升，表明资金关注度增加。关键位置出现放量，技术信号的可靠性增强 |
| **量比萎缩** | 成交清淡，市场参与度低。缩量状态下的技术信号可靠性存疑，需降低预期 |
| **高位异常放量** | 价格高位伴随异常放量，需警惕资金分歧加大 |

**注意事项**：盘中分析时量比可能失真，仅作为辅助参考；收盘后的量比数据更具参考价值。

## 3. 交易决策评估体系

不做"满足条件即执行"的机械匹配，而是基于以下维度形成**信号强度评估**：

### 买入倾向评估维度

1. **趋势阶段**（MACD）
   - DIF处于零轴上方且整体向上 → 中期趋势支撑做多
   - DIF在零轴附近纠缠 → 震荡市，降低预期
   - DIF处于零轴下方但出现底背离 → 潜在转折期，可小仓位关注

2. **波段位置**（RSI）
   - RSI在50中轴上方，或从超卖区（&lt;30）企稳回升 → 波段位置偏积极
   - RSI处于超买区（&gt;70）且价格加速 → 追高风险大，宜观望
   - RSI在50附近无方向 → 波段位置不明确

3. **时机情绪**（KDJ）
   - J值从极端超卖区（&lt;0）回升，或KDJ在低位形成有效拐头 → 短期情绪转暖，提供择时参考
   - J值处于超买区（&gt;100） → 短期过热，不追高
   - KDJ在中间区域（20-80）频繁波动 → 时机不明确

4. **资金确认**（量比）
   - 关键位置出现放量 → 资金认可度高，信号增强
   - 缩量状态下的技术信号 → 可靠性存疑，需降低仓位或观望

5. **价格结构**
   - 是否处于波段支撑（均线/前低/趋势线）附近
   - 是否突破关键阻力位且站稳

**综合判断**：以上维度共振度越高，买入倾向越强。任何单一维度不支持时，都需降低预期或放弃。

---

### 卖出/风控评估维度

1. **趋势破坏**（MACD）
   - DIF从正区向负区转变，或高位死叉伴随柱状图快速缩短 → 中期趋势可能转空
   - 顶背离形成后，价格跌破短期支撑 → 止盈/减仓倾向

2. **波段过热**（RSI）
   - RSI &gt; 70且出现顶背离，或价格加速但RSI无力 → 波段过热，考虑减仓
   - RSI从高位快速回落至50以下 → 多头力量衰退

3. **短期情绪逆转**（KDJ）
   - J值从极端超买区（&gt;100）拐头向下，或KDJ在高位形成死叉 → 短期情绪转弱
   - 在趋势完好时，KDJ高位死叉可能仅为正常回调，需结合MACD和RSI判断是否趋势终结

4. **止损纪律**
   - 入场前需明确止损逻辑（技术位跌破或固定比例），触及后执行
   - 高波动个股止损范围适当放宽，避免正常洗盘震出
   - 亏损头寸不可因"等待反弹"而拖延止损

5. **止盈策略**
   - 趋势完好时让利润奔跑，以均线或MACD状态作为动态止盈参考
   - 快速拉升后RSI超买+KDJ极端+量能萎缩，可考虑分批止盈

## 4. 大盘研判与动态仓位

**大盘仅用于风险评估和仓位上限参考，不作为个股信号的否决条件。**

| 市场环境 | 判断依据 | 仓位倾向 |
|---------|---------|---------|
| **强势** | 主要指数MACD多头，RSI健康，量能充沛 | 可积极配置，单票上限放宽 |
| **震荡** | 指数无明确方向，板块轮动快 | 控制总仓位，降低单票集中度 |
| **弱势** | 指数MACD空头，RSI持续超卖，量能萎缩 | 严格控仓，以防御和观望为主 |

**动态调整原则**：
- 根据近期市场波动率（指数振幅）调整仓位：高波动期降低杠杆和总仓位
- 根据持仓个股的相关性分散风险：同板块持仓不超过总账户的合理比例
- 保留现金比例，为优质信号预留加仓能力
