# desktop-mode

> Source: [WordPress/desktop-mode](https://github.com/WordPress/desktop-mode) — distributed by [TomeVault](https://tomevault.io)

WP Desktop Mode is a WordPress plugin that turns /wp-admin into a desktop-style interface with movable windows and a dock menu. It’s opt-in per user, doesn’t change core, and fully reverts on deactivation.
