# obsidian-paper-curator

> Tome by [zzzlxhhh](https://github.com/zzzlxhhh/obsidian-paper-curator) — distributed by [TomeVault](https://tomevault.io)

obsidian-paper-curator is an LLM-powered paper reading and knowledge base management tool built on LLM + Obsidian. Simply clip papers with the Obsidian Web Clipper, and three Agent Skills will automatically read, summarize, extract concepts, build cross-references, and maintain the structure — turning scattered papers into a compounding wiki.
