# aria-knowledge

> Source: [mikeprasad/aria-knowledge](https://github.com/mikeprasad/aria-knowledge) — distributed by [TomeVault](https://tomevault.io)

Structured knowledge management plugin for Claude Code. Captures insights, decisions, and feedback during AI-assisted development sessions, stages them in backlogs for human review, and promotes   validated knowledge into durable documents. Enforces decision frameworks at the point of code changes. The LLM captures, the human promotes.

## Gemini CLI Config
