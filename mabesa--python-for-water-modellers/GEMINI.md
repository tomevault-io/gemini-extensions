# Python for Water Modellers - Copilot Instructions

This is a Python tutorial project for water modellers using uv as the package manager.

## Project Context

- **Purpose**: Educational tutorial for water resources engineers and hydrologists
- **Package Manager**: uv (fast, modern Python package management)
- **Target Audience**: Water modellers, hydraulic engineers, hydrologists

## Code Style Guidelines

- Follow PEP 8 style guidelines
- Use descriptive variable names relevant to hydraulic/hydrological engineering
- Include docstrings for all functions with parameter descriptions
- Add comments explaining hydraulic/hydrological formulas and concepts

## Domain-Specific Conventions

- Use SI units (meters, m³/s, etc.) unless otherwise specified
- Document assumptions for hydraulic calculations
- Include references to standard equations (Manning's, Darcy-Weisbach, etc.)
- Provide practical examples relevant to water engineering

## Project Structure

- `src/python_for_water_modellers/`: Main package code
- `tutorials/`: Jupyter notebooks for interactive learning
- Use uv commands: `uv sync`, `uv run`, `uv add`

## Dependencies

Focus on scientific Python stack:
- numpy for numerical computations
- pandas for data analysis
- matplotlib for visualization
- scipy for scientific computing
