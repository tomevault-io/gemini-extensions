# Scavenger — Claude Code Context

Scavenger is an open-source, terminal-first AST dependency graph and session memory engine for Claude Code CLI, written in Rust. It reduces token usage by serving focused "capsules" instead of full files, and persists session memory anchored to code symbols across sessions.
