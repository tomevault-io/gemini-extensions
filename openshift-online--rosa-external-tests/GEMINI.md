## Project Overview

ROSA External Tests — tests run from outside the ROSA service to validate the ROSA (Red Hat OpenShift on AWS) managed service. These are end-to-end tests that interact with the live service APIs.

## Key Information

- External test suite for the ROSA managed service
- Tests interact with production or staging ROSA APIs
- Requires ROSA service credentials and access to run
