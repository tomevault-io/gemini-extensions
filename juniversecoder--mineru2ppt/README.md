# MinerU2PPT

> Source: [JuniverseCoder/MinerU2PPT](https://github.com/JuniverseCoder/MinerU2PPT) — distributed by [TomeVault](https://tomevault.io)

High-fidelity tool to convert PDF files and images into fully editable PowerPoint presentations (.pptx). Unlike basic converters, it uses AI-powered structure extraction (MinerU) to accurately reconstruct text, layouts, and images, ensuring the output is not just a screenshot but a professional, modifiable document.
