## tauro-template

> - Responde siempre en español.

## Instrucciones de idioma
- Responde siempre en español.
- Si necesitas pedir permisos, confirmar acciones o explicar resultados, hazlo en español.
- Toda consulta visible para el usuario, incluyendo solicitudes de aprobacion de herramientas, debe ir en español.
- Si generas documentación, comentarios o mensajes para el usuario, usa español salvo que el archivo o el contexto requieran inglés.

## Instrucciones de archivos
- Crea y edita archivos siempre en UTF-8.
- No uses ANSI, Latin-1 ni Windows-1252 para archivos nuevos o modificados.
- Si detectas problemas de acentos, emojis o caracteres extraños, regraba el archivo en UTF-8 antes de finalizar.

## Instrucciones de contexto del proyecto
- Antes de trabajar en este workspace, lee `PROJECT_CONTEXT.md`.
- Usa `PROJECT_CONTEXT.md` como fuente principal para entender el objetivo del template, el estado actual y las decisiones recientes.
- Si haces cambios relevantes de arquitectura, flujo, componentes compartidos, deploy o convenciones, actualiza también `PROJECT_CONTEXT.md` en la misma tarea.

## Instrucciones de documentación
- Mantén README, guías y notas del proyecto en español, salvo que el repositorio requiera documentación pública en inglés.
- No dejes desactualizado `PROJECT_CONTEXT.md` después de cambios importantes.
