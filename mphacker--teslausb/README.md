# TeslaUSB

> Tome by [mphacker](https://github.com/mphacker/TeslaUSB) — distributed by [TomeVault](https://tomevault.io)

Script to turn Raspberry Pi Zero 2 W into a USB thumb drive for a Tesla. Provides 2 USB drives to support dashcam, custom light shows, and custom lock sounds.  Allows remotely updating the files via WiFi.
