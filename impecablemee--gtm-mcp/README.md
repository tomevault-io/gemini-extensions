# gtm-mcp

> Source: [impecablemee/gtm-mcp](https://github.com/impecablemee/gtm-mcp) — distributed by [TomeVault](https://tomevault.io)

Open-source B2B cold outreach pipeline for Claude Code. One /launch command: find companies (Apollo), classify with AI, extract contacts, write deeply personalized sequences, push to SmartLead. Zero LLM calls inside the server, all reasoning stays in Claude. Self-hosted.

The `GEMINI.md` file in this directory is the project config converted for Gemini CLI.
