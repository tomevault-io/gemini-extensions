# opencode-injection-guard

> Source: [remorses/opencode-injection-guard](https://github.com/remorses/opencode-injection-guard) — distributed by [TomeVault](https://tomevault.io)

Prompt injection detection plugin for OpenCode. Works with any LLM provider. Scans tool call outputs with a judge model before they reach the agent. Alternative to OpenAI Guardrails.
