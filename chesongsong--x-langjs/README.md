# x-langjs

> Source: [chesongsong/x-langjs](https://github.com/chesongsong/x-langjs) — distributed by [TomeVault](https://tomevault.io)

An ANTLR4-based domain-specific language that supports embedding x-lang code blocks in Markdown. After parsing and execution, the code is displayed as custom components by a pluggable rendering engine (any UI library). Suitable for scenarios such as AI dialogue and streaming reports.
