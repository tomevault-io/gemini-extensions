# AGENTS

本项目处于 **spec 与开发并行推进阶段**：以已确认 spec 为准推进实现，并保持 spec 与实现同步。

- 先读 **[docs/spec/00-governance.md](docs/spec/00-governance.md)**（治理规则与编写原则）。
- 再按 **[docs/README.md](docs/README.md)** 索引，按需查阅各 spec。
- 既有代码与旧文档仅供参考，不作为裁决依据。
- spec 存在歧义或遗漏时，先澄清；确认后的必要修正需同步回写 spec。
- 每次完成代码或配置修改后，都应检查是否需要同步更新相关 spec、计划、进度与测试文档。
- 开发与调试在 WSL 环境中进行。
