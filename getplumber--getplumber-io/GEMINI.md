# No Em Dashes

Do not use em dashes (—). Prefer commas, parentheses, or short sentences.

## Examples

**Avoid:**
- "The feature is ready—you can deploy it now."
- "Run the script—it will install dependencies automatically."

**Prefer (comma):**
- "The feature is ready, so you can deploy it now."
- "Run the script; it will install dependencies automatically."

**Prefer (parentheses):**
- "The feature is ready (you can deploy it now)."
- "Run the script (it will install dependencies automatically)."

**Prefer (short sentence):**
- "The feature is ready. You can deploy it now."
- "Run the script. It will install dependencies automatically."

Apply this in comments, docs, README text, UI copy, and any prose you write.
