# weekly-git-summary

> Source: [yinzhenyu-su/weekly-git-summary](https://github.com/yinzhenyu-su/weekly-git-summary) — distributed by [TomeVault](https://tomevault.io)

一个跨平台的 CLI 工具，用于生成 Git 提交记录的周报汇总。支持多种输出格式（文本、JSON、Markdown、 HTML），自动扫描项目目录并提取指定时间范围内的提交历史。

## Gemini CLI Config

The `GEMINI.md` file in this directory is the project config converted for Gemini CLI.
Original source: `copilot-instructions.md` in [yinzhenyu-su/weekly-git-summary](https://github.com/yinzhenyu-su/weekly-git-summary).
