# 로컬 서버 실행

로컬 개발 서버를 시작합니다.

## 수행할 작업

1. 가상환경이 활성화되어 있는지 확인
2. `.env` 파일이 존재하는지 확인
3. `python start_broffice.py` 명령으로 서버 시작 (포트 5020)
4. 서버가 정상 시작되면 http://127.0.0.1:5020 에서 접속 가능

## 주의
실행 전 requirements.txt 변경사항이 있으면 `pip install -r requirements.txt`를 먼저 실행합니다.
