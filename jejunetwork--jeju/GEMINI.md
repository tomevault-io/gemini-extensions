# Bun Conventions

**Use Bun for all operations, not npm/npx:**
- Run: `bun run dev` or `bun scripts/deploy.ts`
- Install: `bun install ethers`
- TypeScript runs directly, no transpilation
- `.env` auto-loaded, no dotenv package needed
