# CLAUDE.md

## Git & PR Guidelines

- **All commit messages and PR content must be written in English.**
- Commit messages: concise, imperative style (e.g., "fix: stabilize subgraph ordering in Activity Stream")
- PR title: short, under 70 characters
- PR body format:
  ```
  ## Summary
  <bullet points describing what changed and why>
  ```
- Do NOT include a "Test plan" section in the PR body.
- Do NOT include "Co-Authored-By" in commit messages.
- Do NOT include any Claude Code attribution (footer, badge, etc.) in PRs or commits.
