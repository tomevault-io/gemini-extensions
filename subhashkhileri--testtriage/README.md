# TestTriage

> Tome by [subhashkhileri](https://github.com/subhashkhileri/TestTriage) — distributed by [TomeVault](https://tomevault.io)

TestTriage is a RAG agent that analyzes nightly Playwright e2e and CI test failures from OpenShift CI pipelines in GCS. It uses AI vision on screenshots, parses logs, and generates root cause analysis with test context, failure reasons, and fix suggestions. It can also file JIRA bugs on request.
