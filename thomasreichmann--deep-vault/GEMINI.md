# UI Conventions (MUI + Tailwind)
- Prefer MUI components with Tailwind utility classes on `slotProps`/wrappers.
- Build shared primitives in `src/components/ui/*` (e.g., `Button`, `Card`, `DataTable`, `Badge`, `EmptyState`).
- All interactive components are keyboard accessible and have visible focus states.
- Dark mode supported via MUI theme; no inline color constants—use theme tokens.
