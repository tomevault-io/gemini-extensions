## Project Context

This is a Vue 3 + TypeScript component library project for creating phone UI mockups. The project should:

1. **Component Design**: Create reusable Vue components that simulate phone appearances
2. **TypeScript Support**: Full TypeScript support with proper type definitions
3. **Vue 3 Features**: Use Composition API, reactive refs, and modern Vue patterns
4. **Component Props**: Support various phone models (iPhone 16 series) and custom configurations
5. **Slot Support**: Allow users to place content inside the phone screen area using Vue slots
6. **CSS-in-JS**: Use Vue's scoped styles for component styling
7. **Build Configuration**: Configure for both library distribution and development

## Key Components

- `PhoneUI.vue`: Main phone mockup component
- Support for iPhone 16/Plus/Pro/Pro Max models
- Custom phone configurations (width, height, colors, buttons)
- Slot-based content insertion

## Development Guidelines

- Use Vue 3 Composition API
- Implement proper TypeScript interfaces for props
- Provide comprehensive prop validation
- Use scoped CSS for styling
- Ensure responsive design principles
- Follow Vue 3 best practices for component development
