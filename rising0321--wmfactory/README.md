# WMFactory

> Tome by [Rising0321](https://github.com/Rising0321/WMFactory) — distributed by [TomeVault](https://tomevault.io)

Unified Operator on Interactive World Model is a unified frontend for interactive world models. It lets users select a model, choose a dataset (e.g., CSGO) or directly upload an image, and immediately explore a playable 3D world with WASD movement and drag-to-rotate camera.
