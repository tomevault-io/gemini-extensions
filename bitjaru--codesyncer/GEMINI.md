## 📦 프로젝트 개요

**CodeSyncer**는 AI 코딩 어시스턴트와 멀티 레포지토리 협업을 위한 CLI 도구입니다.

### 핵심 명령어
```bash
codesyncer init      # 초기화
codesyncer update    # 문서 업데이트
codesyncer watch     # 실시간 파일 감시
codesyncer validate  # 설정 검증
```

### 기술 스택
- TypeScript, Node.js
- Commander.js, Chalk, Inquirer
- Chokidar, Jest
