# Stack Frontend : React, TypeScript & Shadcn/ui

## Frameworks et Langage
- Le frontend est une application **React** (Vite) écrite en **TypeScript**.
- **Ne pas** utiliser de JavaScript simple (`.jsx`). Utiliser `PascalCase.tsx` pour les composants.
- Utiliser des composants fonctionnels et des Hooks. **Ne pas** utiliser de Class Components.

## UI et Styling
- L'UI **DOIT** être construite avec **Shadcn/ui**.
- Le style **DOIT** être fait avec **Tailwind CSS**.
- **Ne pas** utiliser d'autres bibliothèques d'UI (Material-UI, Bootstrap, etc.).
- **Ne pas** écrire de CSS simple ou de Styled-Components.

## Gestion d'état et Données
- Pour l'état global, **toujours** utiliser **Zustand**.
- Pour la récupération de données (data fetching) depuis notre API FastAPI, **toujours** utiliser **TanStack Query (React Query)**.
- Pour les graphiques et visualisations, **toujours** utiliser **Recharts**.
