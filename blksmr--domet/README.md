# domet

> Tome by [blksmr](https://github.com/blksmr/domet) — distributed by [TomeVault](https://tomevault.io)

A lightweight React hook made for scroll-based UIs. Use it for a classic scroll-spy, but also for progress bars, lazy-loading sections, or anything that needs to know which section is in view, every time.
