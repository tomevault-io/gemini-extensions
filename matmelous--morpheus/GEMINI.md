# AGENTS.md

Project-level contribution rules for Morpheus.

## Language conventions

- Commit messages must be written in English.
- Source code identifiers must be in English (variables, functions, classes, constants, file names, and CLI flags).
- Source code comments must be in English.
- Documentation can be bilingual (English and Portuguese).
- For bilingual docs, prefer English first, then Portuguese.

## Scope

These conventions apply to all new changes. Legacy content can be migrated incrementally when touched.
