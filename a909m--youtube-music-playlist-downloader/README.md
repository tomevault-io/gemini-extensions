# youtube-music-playlist-downloader

> Source: [A909M/youtube-music-playlist-downloader](https://github.com/A909M/youtube-music-playlist-downloader) — distributed by [TomeVault](https://tomevault.io)

A Docker-based tool that downloads YouTube Music playlists and converts them to high-quality MP3 files. It preserves metadata (artist, title, album), embeds album artwork, and organizes music by playlist.
