# ai-novel-writer

> Tome by [Jieice](https://github.com/Jieice/ai-novel-writer) — distributed by [TomeVault](https://tomevault.io)

AI小说创作系统 - 基于大模型的自动化小说创作助手  功能特性： - 📝 小说项目管理 - 模板化结构，支持多项目管理 - ✍️ 章节润色检查 - 8条铁律风格统一，第一人称转换 - 🔍 AI率检测 - 支持天眼AI、朱雀AIGC等平台检测 - 📤 一键发布 - 导出txt到番茄小说自动发布目录 - 📖 素材库管理 - 内置90年代资料库和参考小说库  技术栈：Minimax/Claude大模型 + Python脚本 + Markdown格式  适用场景：网文写作、乡土小说创作、AI辅助写作
