## 三大核心功能：写 ✍️ / 查 🔍 / 发 📤

### 写（两阶段流程）

**阶段一：写初稿**
1. **读上下文**（分层次，不盲目全读）:
   - 必读：`99_当前状态卡.md` + 前章最后500字
   - 按需读：涉及设定时查 `人物设定.md`/`时间线.md`/`角色关系.md` 等
   - 每弧开头：加读 `弧线摘要.md` + 对应卷节奏规划
2. **构思**（动笔前必填）:
   - 冲突点 / 主角挫折 / 反派行动 / 金手指代价 / 章末钩子 / 场戏规划
3. **写初稿**（第一人称 + 朴素讲故事模式）:
   - **必须第一人称**（"我"不是"陈大山"）——这是降重的关键
   - 对话独立成段 + 双引号
   - 展示不讲述，禁止硬塞语气词

**阶段二：风格润色**
4. **读风格指南**：`山村小神医/项目文件/风格指南.md`
5. **按8条铁律润色**：
   - ①丢人时刻 ②废话/跑题 ③朋友废话 ④笨情绪 ⑤朴素过渡 ⑥人物出场简略 ⑦节奏不均 ⑧内心独白短
6. **更新状态文件**（按需，不盲更全量）:
   - 必更：`进度追踪.md` / `99_当前状态卡.md` / `伏笔追踪.md`
   - 按需更：`角色状态.md`/`资源账本.md`/`时间线.md` 等（本章涉及的才更新）

### 查（检查只查不改，精修靠模型）

```bash
# 字词级检查
python scripts/chapter_health_check.py --project D:/AI/AI小说创作系统/山村小神医 --chapter <章节号>
# 结构级检查
python scripts/publish_readiness_audit.py --project D:/AI/AI小说创作系统/山村小神医 --chapter <章节号>
# 综合决策
python scripts/rewrite_decider.py --project D:/AI/AI小说创作系统/山村小神医 --chapter <章节号>
```

**AI率检测**（在线平台，半自动）:
- 天眼AI（首选）：https://www.tianyanai.org/ （完全免费，实测有效）
- 朱雀AIGC检测：https://matrix.tencent.com/ai-detect/ai_gen （中文最准95%+，免费）
- 提取文本：`python scripts/ai_detect_online.py --project <路径> --chapter <号>`
- 标准：< 20%优秀 / 20-40%可接受 / > 40%需精修

**精修降重**：第一人称+8条铁律写法（实测有效，见`风格指南.md`）
