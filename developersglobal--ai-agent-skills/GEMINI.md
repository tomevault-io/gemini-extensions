# Gemini Instructions for AI Agent Skills

When working with skills in this repository:

1. **Read the SKILL.md fully** before applying any skill
2. **Follow process steps in order** — do not skip or reorder
3. **Complete verification gates** at each step before proceeding
4. **Surface assumptions** — state what you're assuming before coding
5. **Don't overcomplicate** — minimum code that satisfies the spec

## Available Commands

- `/skill-add` — Scaffold a new skill from the template
- `/skill-validate` — Validate a skill file has all required sections  
- `/skill-apply <name>` — Apply a specific skill to the current task
- `/skill-list` — List all available skills

## Skill Application Protocol

When a skill applies to your current task:
1. State which skill you're applying and why
2. Follow the Process steps exactly
3. Complete each verification check before moving on
4. Report evidence at the end (test output, build logs, etc.)
