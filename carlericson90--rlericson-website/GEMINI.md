# Carl Ericson – Personlig B2B-webbplats

## Projektöversikt
- Sajt: ca.rlericson.se
- GitHub-repo: carlericson90/rlericson-website
- Hosting: GitHub Pages (statisk HTML/CSS)
- Ägare: Carl Ericson, Growth Marketing-student på Berghs

## Syfte med sajten
Generera kvalificerade leads och bokningar för B2B-konsulting
inom growth marketing. Primär conversion: boka samtal / ta kontakt.

## Målgrupp
Svenska B2B-företag som söker hjälp med growth marketing,
SEO eller digital strategi.

## Teknisk stack
- Ren HTML och CSS (inga ramverk, inga JS-bibliotek)
- GitHub Pages kräver .html-extension i sitemap-URLer
- Sitemap: /sitemap.xml
- Robots.txt: /robots.txt

## SEO & GEO-status
- Sitemap och robots.txt på plats
- Canonical tags implementerade
- Hreflang implementerat
- Indexering begärd i Google Search Console och Bing Webmaster Tools
- Pågående fokus: teknisk SEO-audit och GEO/AI SEO-optimering

## Arbetsregler (följ alltid dessa)
1. Ändra aldrig fler filer än nödvändigt per uppgift
2. Håll HTML enkelt, semantiskt och välkommenterat
3. Lägg aldrig till externa bibliotek utan godkännande
4. Testa alltid att sidan fungerar lokalt innan push
5. Behåll befintlig URL-struktur – ändra aldrig URL:er utan
   att först diskutera konsekvenser för SEO
6. Alla commits ska ha en tydlig beskrivning på svenska

## Filstruktur (viktigaste filer)
- index.html        → startsida
- sitemap.xml       → sitemap
- robots.txt        → crawl-direktiv
- /assets/          → bilder och CSS

## Nästa prioriterade uppgifter
1. Fullständig SEO + GEO/AI SEO-audit av befintligt innehåll
2. Skapa nya sidor (efter att audit är klar)
