# mc6.sh
This is my personal website. It's a simple static site built with pure HTML and CSS. It's hosted on GitHub Pages. It's currently under construction.

## Design
The design is supposed to be simple and minimalistic. It's supposed to convey a sense of professionalism with the use of a monospaced font and a dark background. It uses a terminal-like cursor animation to indicate that the user is typing. 

## Features
- Simple and minimalistic design
- Cursor animation
- Monospaced font
- Dark background
- GitHub Pages hosting
- Favicon & manifest (located in the ./assets/favicon/ directory)
- Link to my LinkedIn profile
- Link to send me an email
