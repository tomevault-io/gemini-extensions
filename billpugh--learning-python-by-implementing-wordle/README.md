# learning-python-by-implementing-wordle

> Source: [billpugh/learning-python-by-implementing-wordle](https://github.com/billpugh/learning-python-by-implementing-wordle) — distributed by [TomeVault](https://tomevault.io)

This is an effort to find a way that VSCode and Copilot can be set up in a coaching mode to help students learn python. It assumes students already know Java or C, provides custom instructions so the agent acts as a coach, and disables autocomplete, which ignores custom instructions. The students are to write a python program to  play wordle.

## Gemini CLI Config

The `GEMINI.md` file in this directory is the project config converted for Gemini CLI.
