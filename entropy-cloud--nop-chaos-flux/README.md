NOP Chaos Flux is a modern rewrite of the Baidu AMIS low-code renderer, leveraging the React 19 + TypeScript 5.9 + Zustand + Vite 8 technology stack. It supports schema-driven declarative rendering for designer scenarios such as Flow Designer and Report Designer, and includes built-in debugging and diagnostic tools.
