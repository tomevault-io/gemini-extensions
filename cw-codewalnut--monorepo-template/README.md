# monorepo-template

> Tome by [CW-Codewalnut](https://github.com/CW-Codewalnut/monorepo-template) — distributed by [TomeVault](https://tomevault.io)

A modern Full Stack TypeScript first Monorepo template. Uses Bun, Hono, tRPC, Better Auth, Drizzle, Sqlite, React, React Router, Shadcn UI, Vite, Turborepo, Biome.

## Gemini CLI Config

The `GEMINI.md` file in this directory is the project config converted for Gemini CLI.
Original source: `.cursor/rules/*.mdc` in [CW-Codewalnut/monorepo-template](https://github.com/CW-Codewalnut/monorepo-template).
