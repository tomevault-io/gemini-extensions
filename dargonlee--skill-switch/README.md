# skill-switch

> Tome by [DargonLee](https://github.com/DargonLee/skill-switch) — distributed by [TomeVault](https://tomevault.io)

SkillSwitch 是一款跨平台桌面应用，专为 AI 编程助手的用户打造。它让你能够在一个统一的界面中管理多个 AI   工具（Claude Code、Codex CLI、Gemini CLI、Cursor、Windsurf）的 Skill 文件——包括已安装管理、发现新   Skill、创建自定义 Skill、以及备份与恢复。

## Gemini CLI Config

The `GEMINI.md` file in this directory is the project config converted for Gemini CLI.
Original source: `AGENTS.md` in [DargonLee/skill-switch](https://github.com/DargonLee/skill-switch).
