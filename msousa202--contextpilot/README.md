# ContextPilot

> Tome by [msousa202](https://github.com/msousa202/ContextPilot) — distributed by [TomeVault](https://tomevault.io)

Python middleware that compresses and optimizes LLM context before each API call: lower token cost for apps and agents, with quality scoring and safe fallback. Wraps OpenAI, Anthropic, and Google SDKs with minimal code changes.
