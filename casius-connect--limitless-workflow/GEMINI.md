# Limitless Workflow Attendee Repo

## What this repo is

This is the clean workshop version of the Limitless Workflow.

It is optimized for:
- Oracle AI Database on OCI
- Claude Code command-first usage
- preloaded `agent-memory` and `agent-loop` starter packs
- a simple smoke path and first XP Builder success

## Main attendee goal

By the end of setup, attendees should be able to:

1. connect to Oracle
2. load the topic packs
3. pass the smoke path
4. run:
   - `/xp-builder agent-memory procedural-memory`

## Command path

Main command:

```text
/xp-builder agent-memory procedural-memory
```

Advanced next step:

```text
/xp-debrief agent-memory
/refresh-obsidian
/check-learning-state agent-memory
```

## Important rules

- Do not commit `.env` or wallet files.
- Oracle is the source of truth.
- Obsidian is an optional bonus layer.
- The workshop path should stay focused on the real backend only.
