# Codex Instructions For This Repository

This repository is a public starter kit and playbook for building an LLMWiki-style second brain.

When editing this repo:

1. Keep the public instructions generic. Do not hard-code a private local path like `/Users/...`.
2. Put reusable vault instructions in `templates/`.
3. Keep `README.md` beginner-friendly and step-by-step.
4. Keep examples small, fictional, and safe to publish.
5. Do not add private raw notes, customer data, API keys, transcripts, or personal vault content.

Important files:

- `README.md` - the full public playbook.
- `templates/CLAUDE.md` - starter schema for a user's vault.
- `templates/AGENTS.md` - starter Codex instructions for a user's vault.
- `templates/.claude/commands/` - reusable Claude command templates.
- `examples/` - tiny fictional example vault fragments.

Style:

- Prefer plain language over PKM jargon.
- Make instructions copy-pasteable.
- Use lowercase, hyphenated filenames in examples.
- Use Obsidian wikilinks inside example wiki content.
