# External dependencies: use official documentation

When looking up APIs, behavior, or examples for **third-party or AndroidX/Compose libraries** used in this project:

- **Do** use **official documentation** and published API references (project sites, developer.android.com, Kotlin docs, library GitHub READMEs and doc sites, etc.).
- **Do not** treat **local JAR/AAR files** or **Gradle cache sources** as the primary reference for how to use a dependency.

This applies to explanations, code suggestions, and citations in this repo. If official docs are unclear, prefer linking or summarizing from official sources before inferring from decompiled or cached artifacts.
