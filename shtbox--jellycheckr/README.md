# jellycheckr

> Tome by [shtbox](https://github.com/shtbox/jellycheckr) — distributed by [TomeVault](https://tomevault.io)

Jellyfin plugin that detects continuous playback and shows an "Are you still watching?" prompt. If there is no response, playback is paused automatically. Ideal for preventing overnight binge sessions or unattended playback.

## Gemini CLI Config

The `GEMINI.md` file in this directory is the project config converted for Gemini CLI.
