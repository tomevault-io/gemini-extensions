# Glossa

> Tome by [Glaicer](https://github.com/Glaicer/Glossa) — distributed by [TomeVault](https://tomevault.io)

Seamless speech-to-text app for Ubuntu with GNOME Wayland. A background daemon that uses global hotkeys, cloud STT APIs (Groq/OpenAI), and Wayland-native clipboard integration to type as you speak.
