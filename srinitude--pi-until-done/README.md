# pi-until-done

> Tome by [srinitude](https://github.com/srinitude/pi-until-done) — distributed by [TomeVault](https://tomevault.io)

A Pi extension command that executes raw user intent to completion with strict, verifiable termination. `/until-done` runs until a truthful terminal state: 1) all criteria are objectively satisfied or 2) cannot proceed safely, deterministically, or verifiably. It never stops based on perceived completion or invents success.
