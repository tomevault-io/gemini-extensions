## pomodoro-timer

> 使用 React + TypeScript + Tailwind CSS 开发

# 开发指令

## 项目概述

使用 React + TypeScript + Tailwind CSS 开发

## 开发规范

- 组件要可复用
- 代码要有注释

## 设计要求

- 深色主题（背景 #0a0a0a，文字 #ffffff）
- 使用渐变色作为强调色
- 添加平滑的滚动动画
- 确保移动端适配

## 注意事项

- 保持设计简洁，不要过度设计
- 性能优化：图片使用懒加载
- 确保所有链接可点击
