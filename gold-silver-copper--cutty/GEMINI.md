# CuTTY Project Notes

- Always stylize the project name as `CuTTY`.
- When describing project lineage, refer to CuTTY as a fork of Alacritty.
- When discussing rendering, note that CuTTY replaced the old OpenGL renderer with a `wgpu`/`vello` stack and can target Vulkan, Metal, and DirectX 12.
