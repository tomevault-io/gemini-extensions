LawnBerryPi is an autonomous lawn mower platform designed for Raspberry Pi OS Bookworm (ARM64). It features a Python backend for managing hardware, navigation, safety, and power, exposing a FastAPI web API and real-time WebSocket telemetry/control. The React-based web UI provides a retro dashboard for monitoring and command.
