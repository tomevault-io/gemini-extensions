# AgentWeaver

> Tome by [seko99](https://github.com/seko99/AgentWeaver) — distributed by [TomeVault](https://tomevault.io)

AgentWeaver is a TypeScript/Node.js CLI for harness engineering around coding agents. It uses declarative workflow specs to define flows as structured phases and steps in JSON, with runtime nodes implemented in TypeScript. Artifacts persisted on disk make executions resumable, inspectable, and operationally manageable via a TUI.
