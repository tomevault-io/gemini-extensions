# AI Vim Golf Arena

## Overview
Multi-model AI coding challenge arena. Models compete on vim golf challenges.

## Commands
```bash
npm run dev       # Vite dev server
npm run build     # Production build
```

## Tech
- Vite + React 19 + TypeScript
- AI SDK (Anthropic, OpenAI, Google, DeepSeek, Mistral, xAI)
- Radix UI, Tailwind CSS
- Upstash Redis
