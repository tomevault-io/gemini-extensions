# Git 提交与合并规范

- 禁止在 master/main 分支直接提交代码并推送。
- 所有代码改动必须在独立分支完成，并通过 Pull Request 合并。
- 分支命名：feat/<topic>、fix/<topic>、refactor/<topic>、docs/<topic>、chore/<topic>
