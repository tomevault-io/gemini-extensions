# Product Overview

**Class Track** is a comprehensive student management system designed for educational institutions to manage their core academic operations.

## Core Features

- **Student Management**: Complete student profiles with personal information, course enrollment, and academic tracking
- **Teacher Management**: Teacher profiles with specialties, course assignments, and Excel export functionality  
- **Course Management**: Course creation and management with teacher assignments and student enrollment
- **Grade Tracking**: Student grade management with term-based grading system
- **Attendance System**: Daily attendance tracking for students
- **Role-Based Access Control**: Built-in permission system for different user roles
- **Data Export**: Excel export functionality for teachers and other resources

## Target Users

- **Administrators**: Full system access for managing all entities
- **Teachers**: Course and student management within their assignments
- **Students**: View their own academic information and progress

## Key Business Logic

- Students can be enrolled in multiple courses
- Teachers can be assigned to multiple courses
- Grades are tracked by term and course
- Attendance is recorded daily per student
- Role-based permissions control access to different features
