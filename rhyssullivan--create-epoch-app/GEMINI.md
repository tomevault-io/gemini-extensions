## always-use-bun

> Always use `bun` for package management:


Always use `bun` for package management:
- `bun install` (not npm/yarn/pnpm install)
- `bun run <script>` (not npm run/yarn/pnpm run)
- `bunx` (not npx)
- Prefer `bun typecheck` over running `tsc` directly as it uses TypeScript Go and is faster
