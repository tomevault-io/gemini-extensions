# Tilly

Local-first PWA for personal CRM, built with Jazz and AI SDK.

## Essentials

- **Package manager:** `bun` (never npm/pnpm)
- **Typecheck:** `bun check`
- **Build:** `bun build`
- **Never run `bun dev`** - use `bun check` for validation
