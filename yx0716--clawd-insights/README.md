A fork of `rullerzhou-afk/clawd-on-desk` that keeps the real-time desktop pet experience and adds a local insights layer on top. It tracks agent activity on screen, scans local conversation history, and generates AI summaries of what each session was trying to achieve and what it actually accomplished.
