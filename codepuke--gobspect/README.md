# gobspect

> Tome by [codepuke](https://github.com/codepuke/gobspect) — distributed by [TomeVault](https://tomevault.io)

A decode-only introspection library for Go’s encoding/gob format. Inspect gob streams into a structured AST without original types. Includes CLI & query tools.
