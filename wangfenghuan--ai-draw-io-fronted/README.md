# IntelliDraw - AI 驱动的智能图表协作平台  IntelliDraw 是一款智能图表创作平台，利用 AI 技术将自然语言描述即时转换为专业图表，支持流程图、架构图、UML、思维导图等多种类型。  ## 核心亮点  🤖 __AI 智能生成__ - 用自然语言描述你的需求，AI 即刻为你创建图表。支持 Claude、GPT-4/5、Gemini、DeepSeek、GLM、通义千问等 15+ 种 AI 提供商。  👥 __实时协作__ - 多人同时编辑，基于 Yjs CRDT 技术实现无冲突同步。一键分享链接邀请协作者。  🏠 __团队空间__ - 在团队空间中组织图表
