# Project: tenhou-to-mjai

## Critical Policy: Game Log Data

**DO NOT** include, reference, or document any Mahjong Soul (雀魂/Majsoul) game log datasets, archives, download counts, or game statistics in any public-facing files (README, docs, etc.).

- **Tools are fine**: Documenting Majsoul download/convert tools and their usage is allowed.
- **Data is not**: Never mention specific game counts, archive sizes, or validated dataset statistics for Majsoul data.
- **Reason**: Majsoul's Terms of Service prohibit redistribution of game logs. We provide tools for users to download their own data — we do not distribute it.

### What's OK
- "Use `majsoul raw-download` to download game records"
- "The `convert-raw` command converts .pb files to MJAI format"
- Documenting CLI flags, usage examples, pipeline steps

### What's NOT OK
- "3.3M Jade room games"
- "822K Throne room games"  
- Any dataset size stats for Majsoul data
- Any mention of pre-built Majsoul archives or datasets

## Tenhou Data
Tenhou houou dataset stats and distribution are acceptable — the data is published in GitHub Releases under CC BY 4.0.
