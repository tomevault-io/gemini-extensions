# breakster

> Source: [ffoerster/breakster](https://github.com/ffoerster/breakster) — distributed by [TomeVault](https://tomevault.io)

Breakster is a real-time multiplayer music party game. Players listen to short preview clips and try to place each track in the correct chronological position on their personal timeline.
