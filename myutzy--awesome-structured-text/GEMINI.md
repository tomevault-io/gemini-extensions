## Project Overview

This is an [awesome list](https://awesome.re) — a curated collection of resources for IEC 61131-3 Structured Text (ST) programming. The entire list lives in `README.md`. There is no build system, no tests, and no application code.

## Editing Conventions

- Link format: `[Resource Name](link) - Description.` (note the trailing period)
- Use AP-style [title-casing](https://capitalizemytitle.com/) for resource names
- Add links in alphabetical order within their category
- Keep descriptions short, starting with a capital letter
- Remove trailing whitespace
- License: CC0-1.0

## Git
- Never commit directly to `main` - always use a separate branch and pull request
- Omit Claude attribution from commit messages
