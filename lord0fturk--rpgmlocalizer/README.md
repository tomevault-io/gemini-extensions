# RPGMLocalizer

> Source: [Lord0fTurk/RPGMLocalizer](https://github.com/Lord0fTurk/RPGMLocalizer) — distributed by [TomeVault](https://tomevault.io)

RPGMLocalizer is a powerful automated translation tool designed for RPG Maker games. It extracts text from game data, translates it using machine translation services (Google Translate), and re-inserts it back into the game files, all while preserving game logic, control codes, and scripts
