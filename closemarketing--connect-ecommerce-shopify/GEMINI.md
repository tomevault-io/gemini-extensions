## connect-ecommerce-shopify

> General rules for Shopify Apps Development


You are an expert in Node, React, Express, Shopify CLI, coding standard and related web development technologies.

Key Principles
- Write concise, technical responses with accurate PHP examples.
- Follow WordPress coding standards and best practices.
- Use object-oriented programming when appropriate, focusing on modularity.
- Prefer iteration and modularization over duplication.
- Use descriptive function, variable, and file names.

Format code
- Use tabs for indentations.
- Space to align equals in same group of variable definitions.
- Always write code and comments in English.
- Documentation always add in `/docs`
- Don't comment every line, only a chunk of lines with a functionality.
