# Rowbound

Open-source CLI and MCP server for Google Sheets enrichment and data pipelines.

## gstack

Use the `/browse` skill from gstack for all web browsing. Never use `mcp__claude-in-chrome__*` tools.

### Available gstack skills

| Skill | Purpose |
|-------|---------|
| `/office-hours` | Brainstorm and discuss ideas |
| `/plan-ceo-review` | Strategy and CEO-level plan review |
| `/plan-eng-review` | Architecture and engineering plan review |
| `/plan-design-review` | Design plan review |
| `/design-consultation` | Design consultation and feedback |
| `/review` | Code review |
| `/ship` | Ship and deploy |
| `/land-and-deploy` | Land changes and deploy |
| `/canary` | Canary deployment |
| `/benchmark` | Performance benchmarking |
| `/browse` | Web browsing (use this for all web browsing) |
| `/qa` | Quality assurance testing |
| `/qa-only` | QA only (no fixes) |
| `/design-review` | Visual design audit |
| `/setup-browser-cookies` | Set up browser cookies for testing |
| `/setup-deploy` | Set up deployment |
| `/retro` | Retrospective |
| `/investigate` | Debug and investigate issues |
| `/document-release` | Document a release |
| `/codex` | Get a second opinion via Codex |
| `/cso` | Chief Security Officer review |
| `/autoplan` | Auto-review and planning |
| `/careful` | Production safety checks |
| `/freeze` | Scope freeze (limit edits) |
| `/guard` | Guard against risky changes |
| `/unfreeze` | Remove scope freeze |
| `/gstack-upgrade` | Upgrade gstack |
