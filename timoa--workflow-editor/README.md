# workflow-editor

> Tome by [timoa](https://github.com/timoa/workflow-editor) — distributed by [TomeVault](https://tomevault.io)

A VSCode extension providing a visual editor for GitHub Actions workflow files. Open a workflow (YAML), view jobs and steps as a diagram, edit job properties in a side panel, and save back to YAML.

## Gemini CLI Config

The `GEMINI.md` file in this directory is the project config converted for Gemini CLI.
Original source: `.cursor/rules/*.mdc` in [timoa/workflow-editor](https://github.com/timoa/workflow-editor).
