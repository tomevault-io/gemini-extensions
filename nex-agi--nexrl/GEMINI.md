# AI Assistant Instructions for NexRL Project

All AI assistants (Claude, GPT, etc.) working on the NexRL project must adhere to the instructions defined under the `.ai-instructions/` directory. These rules provide the authoritative procedures and policies for your workflow.

Note that for development tasks, you should follow the instructions in `.ai-instructions/developing/*` first,
while for code review tasks, you should follow the instructions in `.ai-instructions/code-review/*`.
