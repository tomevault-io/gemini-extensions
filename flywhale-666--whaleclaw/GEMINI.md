# 中文交互规则

- **用中文思考和回复**：所有面向用户的回复、解释、说明均使用中文
- 代码注释：非显而易见的逻辑注释使用中文
- 变量名/函数名/类名：保持英文（遵循 Python 命名规范）
- 日志/错误信息：面向用户的用中文，内部调试日志用英文
- Git 提交信息：使用英文（遵循项目提交规范）
