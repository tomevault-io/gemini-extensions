## Generation Pipeline (2026-04-19 new)

```
Vault textbook chapter note (ref/books/{edition}/{chXX}/*.md)
  ↓ note has frontmatter `publish: true` + `publish_to: textbook-notes`
  ↓ textbook-share-sync.py (launchd, hm4)
  ↓ extract `## KEY TAKEAWAYS` + `## TEACHING SLIDES`
  ↓ write to textbook-notes/takeaways/{book_slug}/{chapter_slug}.md + slides/{book_slug}/{chapter_slug}.md
  ↓ repos-auto-push.sh (cron 10min) → GitHub
  ↓ GitHub Actions: Marp → .html; pandoc index.md → index.html
  ↓ GitHub Pages deploy
```

**Retired** (2026-04-19): `brenner-slides-sync.py` hardcoded Brenner+Daugirdas — superseded by frontmatter-driven `textbook-share-sync.py`.
