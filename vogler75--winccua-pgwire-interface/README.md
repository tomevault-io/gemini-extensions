# winccua-pgwire-interface

> Tome by [vogler75](https://github.com/vogler75/winccua-pgwire-interface) — distributed by [TomeVault](https://tomevault.io)

A PostgreSQL wire protocol gateway that translates SQL queries to GraphQL calls for WinCC Unified systems. This enables standard SQL clients like psql and other PostgreSQL-compatible tools to query WinCC Unified data using familiar SQL syntax.
